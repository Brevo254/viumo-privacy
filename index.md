# Privacy Policy for Viumo

Last updated: May 2026

Viumo respects your privacy. This application does not collect personal information such as names, emails, contacts, or location data.

## Analytics

Viumo uses anonymous analytics to understand app performance, detect crashes, and improve user experience. This information is not used to identify any individual user.

## Internet Usage

The app requires an internet connection to stream video content. No personal data is stored during this process.

## Data Sharing

Viumo does not sell, trade, or share user data with any third parties.

## Purpose of Data

Any anonymous technical data collected is strictly used to:
- Improve playback performance
- Fix crashes and bugs
- Enhance overall app stability
